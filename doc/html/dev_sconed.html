sconed - SconeServer Daemon

The executable daemon sconed is run in order to start SconeServer. It does a few things, including:

  • Performing platform-specific startup tasks. For example, making the process a daemon on UNIX-like systems, or a system service on Win32.
  • Handling command line options.
  • Creating the root module sconeserver, which loads its configuration file, along with any sub-modules, to get SconeServer into a state where it is ready to run.
  • Passing control to the sconeserver module's event loop.
