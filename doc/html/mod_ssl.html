SSL - Secure Sockets Layer

The ssl module provides stream encryption/decrypion using the OpenSSL library (http://www.openssl.org).

Connections

router.<route-name>.add ( "ssl", channel-name : string )

A connection to the ssl module will insert a Secure Sockets Layer encryption/decryption object into the connection stream at this point, encrypting any outgoing data, decrypting any incoming data. Multiple channels are supported by the module, allowing different encryption settings. The channel name should be passed as an argument with the connection.

SconeScript Interface

In addition to the standard module interface, it also has the following:

add ( channel-name : string )

Add a new SSL channel.

remove ( channel-name : string )

Remove a SSL channel.

list : string

Lists all SSL channels.

<channel-name> : object

Lookup SSL channel given by <channel-name>.

SSL Channels

load_key ( key-name : string )

Load the specified key to use with this SSL channel.
