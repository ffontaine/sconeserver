SconeServer

The sconeserver module is the root of the module tree, and is always present when SconeServer is running. Its main task is to manage the simultaneous handling of client connections, dispatching events to the relevant modules where appropriate.

Configuration

The sconeserver module's configuration file is used to:

  • Set any global options.
  • Load the main modules.

SconeScript Interface

In addition to the standard module interface, it also has the following:

restart()

Restart SconeServer.

shutdown()

Shutdown SconeServer.
