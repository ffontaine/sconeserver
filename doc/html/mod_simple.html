simple - Simple TCP/IP services

The simple module provides a set of standard simple TCP/IP services traditionally supported by UNIX-like systems. These are rarely used, except for debugging purposes.

Connections

router.<route-name>.add ( "simple", service : string )

A connection to the simple module will insert a protocol handler stream for the requested service. If no service name is specified, the module will attempt to determine the service required based on the incoming port number and the operating systems service mappings. If the service is not supported or not recognised, the connection will be dropped.

Supported Services

echo

Implements the echo service as defined in RFC862.

discard

Implements the discard service as defined in RFC863.

daytime

Implements the daytime service as defined in RFC867.

chargen

Implements the chargen service as defined in RFC864.

time

Implements the time service as defined in RFC868.

SconeScript Interface

None.
