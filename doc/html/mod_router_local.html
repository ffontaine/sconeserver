local - Local (UNIX Domain) sockets

The local module allows local socket addresses, which can be used for communicating between processes on the local machine.

SconeScript Interface

In addition to the standard module interface, it also has the following:

addr ( path : string )

Create a socket address for the specified local path.

path should be an absolute path to the socket on the local machine.

If the address is required for listening, then path should not already exist. If it does, then the file will be removed before listening takes place.

Appropriate filesystem permissions are required for the path specified.
