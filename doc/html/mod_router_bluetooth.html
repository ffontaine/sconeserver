bluetooth - Bluetooth sockets

The bluetooth module allows Bluetooth socket addresses, commonly used for connecting mobile devices and accessories, as well as computers.

NOTE: Only rfcomm (stream) socket types are supported at present.

SconeScript Interface

In addition to the standard module interface, it also has the following:

addr ( address : string , channel : int )

Create a socket address for the specified Bluetooth address and channel.

address can be:

  • A hexadecimal Bluetooth device address in the form "aa:bb:cc:dd:ee:ff".
  • An asterisk "*" - indicating a wildcard address. This is only valid when used with listening sockets and indicates that connections should be accepted on all available local devices.

channel should be a Bluetooth channel number.

If the address is required for listening, then host should specify a local device (or, more commonly, "*" for all) and channel should specify a currently unused channel.
